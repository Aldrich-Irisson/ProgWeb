Ejercicios de autoevaluación

3.1 Complete las siguientes oraciones:


  1. El tipo input color permite al usuario introducir un color. Al momento de escribir este libro, la mayoría de los navegadores despliegan el tipo input color como un campo de texto en donde el usuario puede introducir un ___código hexadecimal___ .

  2. El atributo ___placeholder___ nos permite colocar texto temporal en un campo de texto.

  3. Si desea anular la validación, puede agregar el atributo formnovalidate al tipo input ___submit___ .

  4. El atributo ___required___ obliga al usuario a introducir un valor antes de enviar el formulario.

  5. Por lo general se muestra el control ___spinner___ para el tipo input number e incluye sólo los números válidos.

  6. El tipo input ___time___ permite al usuario introducir una hora, minuto, segundo y fracción de segundo.

  7. El elemento ___datalist___ proporciona opciones de entrada para un demento input text.

  8. El elemento ___article___ describe contenido que está separado dd contenido principal de la página y que posiblemente podría utilizarse o distribuirse en otra parte, como un artículo de noticias, mensaje de foro o entrada de blog.

  9. El elemento ___footer___ describe el texto que por lo general aparece en la parte inferior del contenido o de un elemento section.

  10. El elemento ___wbr___ indica el lugar apropiado para dividir una palabra cuando el texto se ajusta en varias líneas.
