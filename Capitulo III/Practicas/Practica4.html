3.4 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.


  1. Los navegadores que despliegan el tipo input color como campo de texto requieren que el usuario introduzca el nombre de un color.

    2. R: Falso. Requieren el valor hexadecimal del color o que seleccionen el color.


  2. Cuando un usuario introduce datos en un formulario y luego lo envía (por lo general, al hacer clic en el botón Enviar), el navegador verifica de inmediato que los datos sean correctos.

    3. R: Verdadero.


  3. HTML5 puede validar si la dirección de correo electrónico introducida por el usuario existe en realidad.

    4. R: Falso. Solo se encarga de validar que cuente con el elemento @


  4. Puede agregar required a cualquiera de los tipos input.

    5. R: Falso. Solo en los input donde el usuario deba de ingresar datos.


  5. Puede habilitar autocomplete sólo para ciertos elementos input.

    6. R: Verdadero.


  6. El elemento time nos permite identificar una fecha, hora o ambas.

    7. R: Verdadero.


  7. El elemento caption proporciona una leyenda para la imagen en un elemento figure.

    8. R: Falso. Es el elemento figcaption.


  8. El elemento details muestra una flecha que apunta a la derecha enseguida de un resumen o leyenda cuando el documento se despliega en un navegador. Al hacer clic en la flecha, ésta apunta hacia abajo y revela el contenido del elemento summary.

    9. R: Falso.Es summary el que muestra la flecha y lo que rebela es el contenido del elemento details.


  9. El elemento footer describe el contenido que por lo general aparece en la parte inferior del contenido o del elemento section.

    10. R: Falso. Describe el pie de página.


  10. El elemento highlight nos permite resaltar texto.

    11. R: Falso. Es el elemento mark.
