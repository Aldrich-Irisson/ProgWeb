Ejercicios de autoevaluación

2.2 Complete las siguientes oraciones:


  1. El elemento ___hr___ inserta una regla horizontal.

  2. Para marcar un superíndice se utiliza el elemento ___sup___ y para marcar un subíndice se utiliza el elemento ___sub___.

  3. El elemento de encabezado menos importante es ___h6___ y el elemento de encabezado más importante es ___h1___.

  4. El elemento ___ul___ marca una lista desordenada.

  5. El elemento ___p___ marca un párrafo.

  6. El atributo ___type = "reset"___ en un elemento input inserta un botón que, cuando se presiona, restablece el contenido del formulario.

  7. El elemento ___tr___ marca una fila de una tabla.

  8. Los ___asteriscos___ se usan por lo general como caracteres de enmascarado en un cuadro de contraseña.
